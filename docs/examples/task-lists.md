# Task Lists

GitHub-style task list items (`- [x]` / `- [ ]`) are converted to Unicode
checkboxes so they render cleanly in Confluence.

- `- [x]` → ✅
- `- [ ]` → ☐

## Release checklist

- [x] Unit tests passing
- [x] Integration tests passing
- [x] CHANGELOG updated
- [x] Version bumped in `pyproject.toml`
- [ ] PyPI release published
- [ ] Announcement posted to Slack `#engineering`
- [ ] README badges updated

## Onboarding steps

- [x] Access to Confluence granted
- [x] Local Python environment set up

- [ ] Test page published to sandbox space
- [ ] Team walkthrough scheduled
