# Security Policy

This document outlines the recommended procedures for reporting security vulnerabilities in the Python repository template.

## Reporting Vulnerabilities

If you find a security vulnerability in [Project Name], please report it to us as soon as possible.

*   **Do not:**
    *   Publicly disclose the vulnerability.
    *   Attempt to exploit the vulnerability.
*   **Please include:**
    *   A description of the vulnerability.
    *   The steps to reproduce the vulnerability.
    *   The impact of the vulnerability.
    *   Any relevant information.

## Contact Information

You can report vulnerabilities to DLSecOps@newday.co.uk

## Security Policy

We will respond to security reports promptly and work to fix the vulnerability as quickly as possible.
